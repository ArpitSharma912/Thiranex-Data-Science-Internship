# Exploratory Data Analysis (EDA) - Titanic Dataset

## Project Objective
The goal of this project is to explore the Titanic dataset and identify patterns, relationships, and insights using statistical analysis and data visualization.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Tasks
- Dataset Overview
- Statistical Summary
- Missing Value Analysis
- Univariate Analysis
- Bivariate Analysis
- Multivariate Analysis
