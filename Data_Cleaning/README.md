# Titanic Data Cleaning and Visualization

## Project Overview
This project focuses on cleaning and visualizing the Titanic dataset. The dataset was preprocessed by handling missing values, removing unnecessary columns, and creating visualizations to understand passenger demographics and survival patterns.

## Dataset
- Titanic Dataset (Kaggle)

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Steps
1. Data Loading
2. Data Exploration
3. Data Cleaning
4. Data Visualization
5. Outlier Detection

## Visualizations
- Age Distribution
- Survival Count
- Gender Distribution
- Survival by Gender
- Passenger Class Distribution
- Fare Distribution
- Correlation Heatmap
- Fare Boxplot
- Age Boxplot
- Pairplot

## Key Findings
- Most passengers were between 20 and 40 years old.
- Female passengers had a higher survival rate than male passengers.
- Third-class passengers formed the largest group.
- Fare contained several high-value outliers.

## Conclusion
The dataset was successfully cleaned and analyzed. The project demonstrates essential data preprocessing and visualization techniques used in data science.
